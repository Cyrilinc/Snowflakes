Class sfUUID

Description of sfUUID Version 3 and 5 sfUUIDs are named based. They require a namespace (another valid sfUUID) and a value (the name). Given the same namespace and name, the output is always the same. Version 4 sfUUIDs are pseudo-random. sfUUIDs generated below validates using OSSP sfUUID Tool, and output for named-based sfUUIDs are exactly the same. This is a pure PHP implementation.

Usage: Named-based sfUUID. $v3uuid = sfUUID::v3('1546058f-5a25-4334-85ae-e68f2a44bbaf', 'SomeRandomString'); $v5uuid = sfUUID::v5('1546058f-5a25-4334-85ae-e68f2a44bbaf', 'SomeRandomString');

Pseudo-random sfUUID $v4uuid = sfUUID::v4();
